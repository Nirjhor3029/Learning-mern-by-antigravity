# 🚀 MERN Stack Learning Plan
> **Goal:** Beginner → Job-Ready Advanced MERN Developer
> **Background:** Laravel + MySQL developer
> **Structure:** 1 Class per day | Code + Explanation + Interview Questions every class

---

## 📌 How This Works
- Every day you come and say: *"Class X shekao"*
- I will create `class-XX/` folder with full content
- After completing a class, mark it `[x]`
- Each class folder will have:
  - `README.md` → Today's Topics + Concepts + Explanations
  - `*.js` / `*.jsx` / project files → Professional-level code
  - Interview Q&A section inside README
